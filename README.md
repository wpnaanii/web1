# BMI Calculator

This is a BMI Calculator web application built with Node.js and Express.

## Features
- BMI calculation
- BMI category detection
- Fat index and muscle index calculation
- Health recommendations
- Styled UI with CSS

## Additional Tasks
As part of the additional assignment requirements, the following tasks were implemented using basic Node.js modules:

1. **Perimeter and Square Calculation**
   - A custom module was created to calculate the perimeter and area of a geometric shape.
   - Functions are exported using `module.exports` and imported with `require()`.

2. **Minimum and Maximum Value Finder**
   - A separate module was implemented to find the minimum and maximum values in an array of numbers.
   - Demonstrates working with arrays and built-in JavaScript functions.

## Technologies
- Node.js
- Express
- HTML
- CSS

## How to run
1. Install dependencies: npm install
2. Run the server: node app.js
3. Open in browser: http://localhost:3000
